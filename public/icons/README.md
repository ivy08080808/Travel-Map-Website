# 圖標下載指南

請從 Flaticon 下載以下圖標（alt 系列），並將 SVG 文件放在此目錄：

## 需要下載的圖標：

1. **飛機** (plane-alt): https://www.flaticon.com/free-icon-font/plane-alt_7433955
   - 文件名：`airplane-alt.svg` 或 `airplane.svg`

2. **巴士** (bus-alt): https://www.flaticon.com/free-icon-font/bus-alt_7433780
   - 文件名：`bus-alt.svg` 或 `bus.svg`

3. **汽車** (car-alt): 請在 Flaticon 搜索 "car-alt"
   - 文件名：`car-alt.svg` 或 `car.svg`

4. **步行** (walking-alt): 請在 Flaticon 搜索 "walking-alt"
   - 文件名：`walking-alt.svg` 或 `walking.svg`

5. **火車** (train-alt): 請在 Flaticon 搜索 "train-alt"
   - 文件名：`train-alt.svg` 或 `train.svg`

6. **船** (boat-alt): 請在 Flaticon 搜索 "boat-alt"
   - 文件名：`boat-alt.svg` 或 `boat.svg`

## 下載步驟：

1. 訪問 Flaticon 網站
2. 點擊 "Download" 按鈕
3. 選擇 "SVG" 格式
4. 將下載的文件重命名並放到 `public/icons/` 目錄

## 注意：

- 確保所有圖標都是同一系列（alt 系列）
- 圖標應該是素色線條風格
- 如果找不到某個圖標的 alt 版本，可以使用類似的圖標
